# Streamore — Privacy Policy

This repository hosts the privacy policy for the **Streamore** app (iPhone, iPad, Apple TV)
via GitHub Pages.

📄 **Live page:** https://nicolajhansen97.github.io/streamore-privacy/

Streamore is a "bring your own media" player: it ships with no content and connects only
to sources the user configures. It runs no developer-side server and includes no analytics,
ads, or tracking.

To update the policy, edit `index.html` and push to `main`; GitHub Pages redeploys
automatically.
